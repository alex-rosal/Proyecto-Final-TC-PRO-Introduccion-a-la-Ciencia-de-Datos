Proyecto Final - Alexandra Rosal

Como proyecto final del curso de Introducción a Ciencia de Datos con Tecnolochicas, entrego una aplicación móvil desarrollada con MIT App Inventor 2, que analiza un conjunto de datos históricos relacionados con las condiciones de clima del Gran Premio de Gran Bretaña de la Fórmula 1. Directamente, la app permite visualizar, detectar anomalías y analizar datos correspondientes a la velocidad del viento (en millas por hora), y temperatura (en grados Celsius), permitiendo identificar tendencias y variaciones de datos a lo largo del tiempo para generar una predicción o conclusión utilizando inteligencia artificial como herramienta.

El dataset utilizado incluye:

Año de la Carrera
Velocidades de viento (mph)
Temperatura
Precipitación

Para este proyecto, como el GP de Gran Bretaña se realiza una vez al año, se toma la información de las condiciones meteorológicas del momento en el que se realizó la carrera.

Las preguntas de investigación fueron:
¿En qué año podía haber la menor velocidad del viento?
¿Cuáles son las implicaciones de estos resultados para las futuras carreras de F1 en el GP de Gran Bretaña después de 2024?
